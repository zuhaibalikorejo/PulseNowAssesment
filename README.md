# PulseNowAssesment

A Flutter mobile application that displays real-time market data for cryptocurrency symbols, including current prices and 24-hour price changes.
Overview
This application provides users with up-to-date cryptocurrency market information in a clean, user-friendly interface.

# Features
Display a list of cryptocurrency symbols
Show current prices for each symbol
Display 24-hour price changes
Real-time market data updates


# Technologies
Flutter - Cross-platform mobile framework
Kotlin - Android native code
Java - Android compatibility
Gradle - Build automation

# Getting Started
Prerequisites
Flutter SDK installed
Android Studio or VS Code with Flutter plugins
Android device or emulator
